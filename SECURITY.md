# Security Policy

## Security Model

LNDIS AI Asistanım capability-based architecture kullanır.

LLM doğrudan sistem erişimine sahip değildir.
Tüm sistem işlemleri Tool katmanı üzerinden gerçekleştirilir.

## Default Restrictions

- DELETE operations are disabled.
- System directories are write-protected.
- Registry access is blocked.
- Network access is disabled by default.
- No remote access.

## Reporting Vulnerabilities

Please open a security issue or contact the maintainers.
